## Requirements
  * [Redis](https://redis.io/topics/quickstart)

## Installation

```sh
$ cd rate-limiter
```

```sh
$ bundle install
```


## Start the server

```sh
$ foreman start
```

Check out the example response from the link below
```sh
http://localhost:3001/api/v1/users
```

## Tests

```sh
$ rspec --format documentation
```
